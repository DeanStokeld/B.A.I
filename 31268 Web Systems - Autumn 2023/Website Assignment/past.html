Computing Education

During High School in Year 11 and 12 I completed "Information Processes and Technology" (IPT) as one of my subjects during my HSC.

I have also completed a 6-month Data Science and Innovation course at UTS through the Institute of Data last year, this covered many topics including, but not limited to:

  • SQL
  • Data cleaning & visualisation
  • Developing machine learning models (classification/regression)
  • Deep learning (FFNNs, RNNs & CNNs)
  • Reinforcement learning
  • Cloud Computing
  • Deploying ML models

In additon to the list above, I was also tasked to develop machine learning model to detect pneumonia in 5,216 chest x-rays images which yielded a 94.44% accuracy and took 12 hours to train.

I first had to clean the data then analyse using python libraries, such as NumPY, Matploblib and Pandas. I then performed data pre-processing & data augmentation on the data followed by training different models using the Python package 'TensorFlow' at various epochs and batch sizes with a ResNet50V2 model performing with the highest accuracy of 94.44%.
